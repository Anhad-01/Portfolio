# Components

This folder is intentionally modular:

- `Hero.jsx`: intro + primary CTAs
- `About.jsx`: education + CGPA
- `Skills.jsx`: floating/glowing skill pills
- `Experience.jsx`: vertical interactive timeline
- `Projects.jsx`: 3D-tilt cards + links
- `Achievements.jsx`: hackathon highlights
- `Footer.jsx`: contact links

UI primitives live in `components/ui/`.
